UI Importer Pro

Roblox Studio plugin documentation for converting imported, offset-heavy UI into responsive scale-based layouts.

Quick Start

  1. Install or sync the plugin in Roblox Studio.
  2. Select a ScreenGui or supported UI object in Explorer.
  3. Click the FixScale toolbar button.
  4. Review the converted layout across multiple viewport sizes.

Core Behavior

  • Traverses selected UI hierarchies recursively.
  • Converts Size from absolute pixels to parent-relative scale.
  • Converts Position using parent-relative absolute placement.
  • Sets anchors to center while preserving current visual placement.
  • Adds missing UIAspectRatioConstraint instances.
  • Uses Studio history waypoints so conversions can be undone.

API Reference

ScalingEngine.ToScale(instance: Instance): boolean

Converts a single GuiObject to scale-based Size and Position. Returns whether the object was processed.

ScalingEngine.ProcessRecursive(root: Instance): number

Processes root and every descendant. Returns the number of converted UI objects.

FileParser.validate(instance: Instance): boolean

Checks whether an instance can be used as a selected root.

FileParser.identify(instances: { Instance }): { Instance }

Filters a Studio selection down to supported UI roots.

FileParser.processImported(instances: { Instance }): number

Validates selected instances, converts each valid root, and returns the converted object count.
