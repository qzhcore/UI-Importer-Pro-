What It Does

UI Importer Pro cleans up UI that was imported as fixed pixel frames. It keeps the visual layout intact while converting the hierarchy into a better Roblox-native structure.

Scale Conversion

Turns absolute Size, Position, padding, grid cells, and canvas offsets into responsive scale values.

Layout Organization

Detects rows, columns, grids, page stacks, and table-like groups, then adds the best matching Roblox layout object.

Studio Safe

Uses undo waypoints and avoids overriding child positions when a layout object should control placement.

AutoLayout Pass

Before scaling, the organizer scans selected UI containers and applies layout helpers only when the geometry is clear enough to infer safely.

UIListLayout

Applied to aligned horizontal or vertical child groups, with padding inferred from the average gap.

UIGridLayout

Applied to uniform matrix groups with consistent row and column counts.

UITableLayout

Applied to table-like row and column groups where children are aligned but not uniformly sized.

UIPageLayout

Applied when children look like page-sized panels inside the same container.

UIScale

Local scale objects are accounted for when calculating the final responsive size.

ScrollingFrame

CanvasSize offsets are converted relative to the frame's own absolute size.

API Reference

The plugin is modular so the organizer, parser, and scaler can evolve independently.

LayoutOrganizer.ProcessRecursive(root: Instance): number

Infers and inserts layout objects for supported UI containers.

ScalingEngine.ProcessRecursive(root: Instance): number

Converts supported descendants into scale-based responsive sizing.

FileParser.processImported(instances: { Instance }): number

Filters Studio selection, runs the organizer pass, then runs the scaler pass.
